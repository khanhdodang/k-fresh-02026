# 📄 Test Case Document: Compare Products Feature

## TC-CP-002: Verify detailed information in the comparison table (3 products)

###  1. General Information
| Attribute | Details |
| :--- | :--- |
| **Test Case ID** | `TC-CP-002` |
| **Module** | Product Comparison |
| **Priority** | High |
| **Test Type** | Functional Testing |
| **Description** | Verify the workflow of adding 3 products to the comparison list and validate all detailed attributes displayed on the comparison page. |

---

###  2. Pre-conditions
* The user is currently on the category page.
* The **Compare** list is empty (no previous products added).

---

###  3. Test Data
| Variable / Object | Input Value / Details |
| :--- | :--- |
| **Category URL** | `path=57` (Tablets Category) |
| **Product 1** | Name: `HTC Touch HD` (ID: 28) |
| **Product 2** | Name: `Palm Treo Pro` (ID: 29) |
| **Product 3** | Name: `Canon EOS 5D` (ID: 30) |

---

###  4. Test Steps & Expected Results

| Step | Action | Expected Result |
| :---: | :--- | :--- |
| **1** | Navigate to the Tablets category page. | The category page loads successfully. |
| **2** | Click the **"Compare this Product"** button for **HTC Touch HD**. | A toast notification appears displaying: **"Product Compare (1)"**. |
| **3** | Click the **"Compare this Product"** button for **Palm Treo Pro**. | A toast notification appears displaying: **"Product Compare (2)"**. |
| **4** | Click the **"Compare this Product"** button for **Canon EOS 5D**. | A toast notification appears displaying: **"Product Compare (3)"**. |
| **5** | Navigate to the URL: `/index.php?route=product/compare`. | The **Product Comparison** page loads successfully. |
| **6** | Verify the number of product columns in the comparison table. | The table displays exactly **3 columns** for the products. |
| **7** | Verify the **"Product"** row. | Displays the names: `HTC Touch HD`, `Palm Treo Pro`, and `Canon EOS 5D`. |
| **8** | Verify the **"Image"** row. | Each product displays a correct thumbnail image. |
| **9** | Verify the **"Price"** row. | Displays: `$146.00` / `$337.99` / `$134.00` respectively. |
| **10** | Verify the **"Model"** row. | Displays: `Product 1` / `Product 2` / `Product 3`. |
| **11** | Verify the **"Brand"** row. | Displays: `HTC` / `Palm` / `Canon`. |
| **12** | Verify the **"Availability"** row. | The correct stock status is displayed for each product. |
| **13** | Verify the **"Rating"** row. | Displays the star rating or the text "Based on N reviews". |
| **14** | Verify the **"Weight"** row. | Displays the correct weight (e.g., HTC: `146.40g` / Palm Treo Pro: `133.00g`). |
| **15** | Verify the **"Dimensions"** row. | Displays the dimensions (L x W x H) for each product. |

---

###  5. Expected Final Result
> The comparison table must successfully display all **3 products** with their respective rows fully populated with accurate data, including: *Product, Image, Price, Model, Brand, Availability, Rating, Summary, Weight, and Dimensions*.
