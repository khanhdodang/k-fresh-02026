# 📄 Test Case Document: Compare Products Feature

## TC-CP-001: Add 2 products to Compare and verify the comparison page

### 1. General Information
| Attribute | Details |
| :--- | :--- |
| **Test Case ID** | `TC-CP-001` |
| **Module** | Product Comparison |
| **Priority** | High |
| **Test Type** | Functional Testing |
| **Description** | Verify the workflow of adding 2 products from the category page to the comparison list and validate the accuracy of the data displayed on the comparison page. |

---

### 2. Pre-conditions
* The browser is open with a stable internet connection.
* The **Compare** list is currently empty (no products have been added yet).

---

### 3. Test Data
| Variable / Object | Input Value / Details |
| :--- | :--- |
| **Category URL** | `https://ecommerce-playground.lambdatest.io/index.php?route=product/category&path=57` |
| **Product 1** | Name: `HTC Touch HD` <br> Price: `$146.00` <br> Brand: `HTC` |
| **Product 2** | Name: `Palm Treo Pro` <br> Price: `$337.99` <br> Brand: `Palm` |

---

### 4. Test Steps & Expected Results

| Step | Action | Expected Result |
| :---: | :--- | :--- |
| **1** | Navigate to the Tablets category URL. | The Tablets page loads successfully and displays the list of products. |
| **2** | Hover over the **HTC Touch HD** product card. | The action icon group (cart, wishlist, compare, quick view) is displayed on the product card. |
| **3** | Click the **"Compare this Product"** button on the HTC Touch HD. | - A toast notification appears: *"You have added HTC Touch HD to your product comparison!"* <br>- The link in the toast displays the correct count: **"Product Compare (1)"**. |
| **4** | Wait for the toast notification to display. | The toast notification disappears automatically after a few seconds. |
| **5** | Hover over the **Palm Treo Pro** product card. | The action icon group is displayed on the product card. |
| **6** | Click the **"Compare this Product"** button on the Palm Treo Pro. | - A toast notification appears: *"You have added Palm Treo Pro to your product comparison!"* <br>- The link in the toast updates the count: **"Product Compare (2)"**. |
| **7** | Click the **"Product Compare (2)"** link in the toast <br>*(Or navigate directly to: `/index.php?route=product/compare`)*. | The system successfully redirects the user to the **Product Comparison** page. |
| **8** | Verify the information displayed on the comparison page. | - The `H1` tag displays: **"Product Comparison"**.<br>- The URL contains the string: `route=product/compare`.<br>- The comparison table correctly shows **2 columns** for HTC Touch HD and Palm Treo Pro.<br>- **Price:** HTC (`$146.00`) / Palm (`$337.99`).<br>- **Brand:** HTC (`HTC`) / Palm (`Palm`).<br>- The table displays all other fields (Model, Availability) correctly.<br>- A **"Remove"** button is present for each product. |
